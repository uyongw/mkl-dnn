Initializes a convolution descriptor conv_desc for forward propagation using prop_kind (possible values are mkldnn_forward_training or mkldnn_forward_inference), alg_kind, memory descriptors, strides, padding_l, padding_r, and padding_kind.

In order to create a convolution without bias, bias_desc should be either NULL or point to a descriptor with memory format equals to mkldnn_format_undef.

Note
if padding_r is NULL, the padding is supposed to be symmetric
memory descriptors are allowed to be initialized with mkldnn_any value of format_kind.
